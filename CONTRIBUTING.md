# Contributing to dotnet-template

Thank you for considering contributing. Please follow these guidelines:

- Fork the repository and create your branch from `main`.
- Run `make lint` and `make test` before opening a pull request.
- Add or update tests for behavior changes.
- Update `README.md` or related docs when setup, endpoints, or scripts change.
- Keep security-sensitive findings out of public issues and use `SECURITY.md` instead.
